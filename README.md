# Tubus – Десктопный клиент

**Tubus** — это кроссплатформенное приложение для работы с конструкторской документацией предприятия. Входит в состав программного пакета Tubus. Обеспечивает постоянный доступ к актуальным чертежам и документам в зависимости от должностных прав сотрудника.

🔗 **Репозиторий проекта:** [wert-dbpik/desktop-client](https://github.com/wert-dbpik/desktop-client)

---

## Возможности

### Для инженеров, технологов и руководителей

- 📄 **Просмотр документации** (PDF, PNG, JPG, JPEG, DXF, STEP, EASM)
- 💬 **Комментарии** — текстовые и в виде изображений
- 🧩 **Группировка документов** по изделиям
- ✏️ **Добавление, удаление, изменение и смена жизненного статуса** документов
- 📥 **Импорт Excel-файла изделия** для просмотра чертежей по конкретному изделию
- 📘 **Журнал регистрации** новых номеров документов
- 💬 **Встроенный чат** для переписки

### Для администратора

- 👥 **Управление пользователями** — добавление, удаление, изменение, перенос по группам прав
- 🔐 **Настройка прав групп** (разделение по профессии)
- 📜 **Логи активности** — входы/выходы, изменения документов, удаления, копирования, версии приложения
- 🛠️ **Отслеживание ошибок** в работе десктопного и мобильного приложений

## Установка и запуск

1. Убедитесь, что установлена **Java 8**.
2. Скачайте последнюю версию клиента из раздела [Releases](https://github.com/wert-dbpik/desktop-client/releases).
3. Запустите `.jar` файл:
   ```bash
   java -jar Tubus.jar
При первом запуске укажите адрес сервера Tubus-server.

## Настройка подключения
Серверная часть работает на Java 8 + Spring Boot 2 и использует PostgreSQL.
Клиент подключается к серверу по REST API.
